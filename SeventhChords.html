Seventh Chords

There are five common types of seventh chords, Major 7th, Dominant 7th, Minor 7th, Minor 7b5, Diminished 7th. By definition a major 7th chord contains the notes of the first, third, fifth and seventh note of a major scale. It is common on the guitar to voice the notes in different spacings. This page contains some of the common voicings for seventh chords. Once you have a major seventh chord you can make changes to that structure to become the other types of seventh chords. The following list shows how to move through the different seventh chords of the same root.

  1. Start with any voicing of a major 7th chord.
  2. Lower the seventh of a major 7th to create a dominant 7th chord.
  3. Lower the third of a dominant 7th to create a minor 7th chord.
  4. Lower the fifth of a minor 7th to create a minor 7b5 chord.
  5. Lower the seventh (again) of a minor 7b5 to create a diminished 7th chord.

If these chords are new to you, this list can seem enormous and intimidating. I suggest starting with the root position versions especially the open position voicings. This page contains enough material for a lifetime of learning.

Strings 4-1

The first group of seventh chords has the ROOT on string 4, third on string 3, fifth on string 2, and the seventh on string 1. The major 7 form is quite playable but depending on your hand size, the others are difficult to impossible. This is known as close position voicing, where root, third, fifth, and seventh are in consecutive order within the same octave (close together). It is a comfortable voicing on a keyboard but not as comfortable for the guitar. Some common chords are in close position voicing (like the basic C chord) but most are not. If your hand can reach these shapes you should use them, they are beautiful.

More common than the hand stretcher forms are the open position voicings shown below. The first group moves the third of the chord up an octave. So the voicing from bottom to top is root, fifth, seventh, third. In some corners this is called the "drop 2" voicing, because the second note of the 4 note chord (root,third,fifth,seventh) is 'dropped' towards the floor (moved to string 1). Please understand however that nothing is dropped musically, it is raised an octave. The result is a much more playable family of seventh chords The other rows of chords use a similar idea in opening up the voicing so that not all of the notes are in a single octave.

Notice that due to the symetrical structure of the Diminished 7 chord (consecutive minor thirds or its inversion) all of the different family of inversions end with the same shape for Diminished 7 in all inversions. This means that every Diminished 7 chord is really four chords in one. Great value! Also another fascination attribute of the Diminished 7 chord is that if you lower the root of the chord, you'll create a Dominant 7 chord of that lowered root. In fact if you lower any single note of a Diminished 7 chord you'll create a Dominant 7 chord whose root if the note you just lowered. So even though with our diagrams of the family of seventh chords of the same root, it seemed that the Diminished 7 was three changes away from a Dominant 7 chord, it actually a closer relationship. But that relationship is when the ROOT of the Dominant chord is raised a half step or conversely ANY single note of a Diminished 7 chord is lowered a half step. This relationship is commonly used in jazz.

Strings 6-3

The first group of seventh chords has the ROOT on string 6, fifth on string 5, seventh on string 4, and the third on string 3. (root position). These chords with the root on string six are complemented with another group shown later using strings 6, 4, 3, 2. Many students learn the group using Strings 6, 4, 3, 2 before learning this set. One reason being that these voicings result in lower notes mostly below middle C in real range (guitar is notated an octave higher than it really sounds). The voicings using Strings 6, 4, 3, 2 raise the pitch somewhat and they pair nicely with the voicings using Strings 5, 4, 3, 2.
