Rootless Voicings

Once you have some fluency with the seventh chord voicings, you can add extensions to those chords by replacing either the root or the fifth (or both) with a ninth or thirteenth respectively. These alterations work especially well with the voicings that are inversions (not root position). It is common to add ninths, elevenths and thirteenths to dominant 7th chords. Below show some of the common alterations to the first inversion dominant 7th chord. It is possible to the play the root (marked with the white circle) of many of these chords but they are effective even without the root. Notice that several of the shapes are exactly like seventh chords of a different root. (i.e. rootless C9 = Em7b5, rootless C7b9 = Edim7). Also notice that a b13 is the same note as a #5, different names for the same sound.

(9 replaces root) C7b9
(b9 replaces root) C13
(13 replaces fifth)
(9 replaces root) C13b9
(13 replaces fifth)
(b9 replaces root) C7b9b13
(b13 replaces fifth)
(b9 replaces root) C9b13
(b13 replaces fifth)
(9 replaces root) C7#9

(#5 replaces fifth)
(#9 replaces root) C7#9b5
